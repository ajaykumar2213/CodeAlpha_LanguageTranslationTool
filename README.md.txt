# Language Translation Tool

## Description

This project is a simple Language Translation Tool developed using Python. It allows users to enter text in one language and translate it into another language using Google Translator.

## Features

* Translate text between languages
* Easy-to-use interface
* Fast and accurate translation
* Built with Python

## Technologies Used

* Python
* deep-translator library

## How to Run

1. Install Python
2. Install the required package:
   pip install deep-translator
3. Run the program:
   python translator.py

## Example

Input:
Hello Friend

Output:
வணக்கம் நண்பரே
